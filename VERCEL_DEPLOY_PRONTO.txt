1️⃣  Ir para a pasta:
    cd schulterplan-vercel/

2️⃣  Fazer push para GitHub:
    git init
    git add .
    git commit -m "Initial: SchulterPlan"
    git remote add origin https://github.com/SEU_USUARIO/schulterplan.git
    git push -u origin main

3️⃣  Conectar Vercel ao GitHub:
    - Ir para https://vercel.com
    - Conectar conta GitHub
    - Importar repositório "schulterplan"
    - Clicar "Deploy"

    💡 Vercel atualizará AUTOMATICAMENTE a cada git push

P: Preciso de servidor próprio?
R: Não! Vercel é grátis e hospeda TUDO. Sem custos.

P: Quanto tempo leva o deployment?
R: 1-2 minutos no Vercel. Mais rápido que café ☕

P: Posso atualizar depois?
R: Sim! Cada 'git push' atualiza automaticamente. Zero downtime.

P: E se quebrar algo?
R: Vercel mantém histórico de todas as versões. Reverter com 1 clique.

P: Funciona em mobile/tablet?
R: Sim! Mas desktop 1920×1080 é recomendado (especialmente para dashboard).

P: Os dados do paciente ficam seguros?
R: Sim! Tudo é local (browser). Nada é enviado para servidor.
   planning.json é carregado localmente. 100% privado.
